# Prespotify
software I use to generate an ogg of what my songs sound like on spotify.
basically an offline version of loudness penalty by meterplugs https://www.loudnesspenalty.com/

Audio engineers dislike having to hear spotify compressing their wavs to shitty oggs, so i made a script that gives producers a heads up on what their music might sound like before sending out their loseless files to the distributors.
